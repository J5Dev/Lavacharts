Charts & Configuration


LavaCharts is extensively customizable, utilizing all the configuration of Google Charts

All configuration options closely mirror the javascript structure of the Google Chart API.
The was done to make it easy for looking up references or if you have used the Chart API before.


Currently Supported Charts

  • Line,  Area,  Pie & Donut,  Column,  Geo,  More Coming Soon...

Chart Methods:

  • The Chart methods always return themselves, so options are chainable.
  • The Charts all extend the parent "Chart" class to provide common options to all specific Chart objects.
  • Charts also have specific config methods, to further customize the chart. (use the above links to jump to a section)

Setting up the options for the charts:
