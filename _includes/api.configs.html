Config Objects

All of the configuration objects can be created by one of two ways:

  • Passing an associative array into the constructor.
  • Using the configuration options as methods of the object. (Chainable, see Chaining section)
  • The only exception is the jsDate object, it mimics the javascript date signature.

Background Color

Lava::backgroundColor( array $config );
  • strokestringThe color of the chart border, as an HTML color string.
  • strokeWidthintThe border width, in pixels.
  • fillstringThe chart fill color, as an HTML color string.
  • returnsobject (backgroundColor)backgroundColor Config Object.

Chart Area

Lava::chartArea( array $config );
  • leftintegerHow far to draw the chart from the left border.
  • topintegerHow far to draw the chart from the top border.
  • widthintegerWidth of the chart.
  • heightintegerHeight of the chart.
  • returnsobject (chartArea)chartArea Config Object.

Text Style

Lava::textStyle( array $config );
  • colorstringColor of the text.
  • fontNamestringName of font.
  • fontSizestringSize of font, in pixels.
  • returnsobject (textStyle)textStyle Config Object.

Legend

Lava::legend( array $config );
  • positionstringPosition of the legend.
  • alignmentstringAlignment of the legend.
  • textStyleobject (textStyle)Text style of the legend.
  • returnsobject (legend)legend Config Object.

Tooltip

Lava::tooltip( array $config );
  • showColorCodebooleanShow color code for the tooltip.

  • returnsobject (tooltip)tooltip Config Object.

Color Axis

Lava::colorAxis( array $config );
  • minValueintegerMinimum value for chart color data.
  • maxValueintegerMaximum value for chart color data.
  • valuesarrayControls how values are associated with colors.
  • colorsarrayColors to assign to values in the visualization.
  • returnsobject (colorAxis)colorAxis Config Object.

  • returnsobject (slice)slice Config Object.

Size Axis

Lava::sizeAxis( array $config );
  • maxSizeintegerMaximum radius of the largest possible bubble, in pixels.
  • maxValueintegerThe size value to be mapped to $this->maxSize
  • minSizeintegerMininum radius of the smallest possible bubble, in pixels
  • minValueintegerThe size value to be mapped to $this->minSize
  • returnsobject (sizeAxis)sizeAxis Config Object.

  • returnsobject (jsDate)jsDate Config Object.
