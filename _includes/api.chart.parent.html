Lava::XxxxChart( string $label = '' )

Replace "Xxxx" with any supported chart, no quotes and new chart of that type will be created. (note, use CamelCase)

  • paramstringSets a label for the chart to correspond with a DataTable. (optional, but recommended)
  • returnsobject (Chart)Returns a new chart of specified type for chaining.

setConfig( array $option )
  • paramarraySets configuration options from array of values.
  • returnsobject (Chart)Returns the chart for chaining.

outputInto( string $elementID )
  • elementIDstringSets the ID of the HTML element to output the chart into.
  • returnsscript blockJavascript blocks to insert into the page.

dataTable( DataTable $dataTable | string $label )

If the DataTable was created with a label that doesn't match the Chart label, then pass the label to this function.
Otherwish, if the DataTable was created with no title, then pass a DataTable object to this function to assign to the chart.

  • paramobject (DataTable)Sets the DataTable to use for the chart.
  • ORparamstringSets the labeled DataTable to use for the chart.
  • returnsobject (Chart)Returns the chart for chaining.

backgroundColor( backgroundColor $backgroundColor)
  • paramobject (backgroundColor)Sets the chart background fill color, border and border color.
  • returnsobject (Chart)Returns the chart for chaining.

chartArea( chartArea $chartArea )

Defines where the chart itself is drawn, excluding axis and legends.

  • paramobject (chartArea)An object with members to configure the placement and size of the chart area.
  • returnsobject (Chart)Returns the chart for chaining.

colors( array $colorArray )

The colors to use for the chart elements.

  • paramarrayAn array of strings, where each element is an HTML color string.
  • returnsobject (Chart)Returns the chart for chaining.

fontSize( integer $fontSize )

The default font size, in pixels, of all text in the chart. You can override this using properties for specific chart elements.

  • paramintegerNumber of pixels
  • returnsobject (Chart)Returns the chart for chaining.

fontName( string $fontName )

The default font face for all text in the chart. You can override this using properties for specific chart elements.

  • paramstringValid font name.
  • returnsobject (Chart)Returns the chart for chaining.

height( integer $height )

Height of the chart, in pixels.

  • paramintegerNumber of pixels
  • returnsobject (Chart)Returns the chart for chaining.

width( integer $width )

Width of the chart, in pixels.

  • paramintegerNumber of pixels
  • returnsobject (Chart)Returns the chart for chaining.
