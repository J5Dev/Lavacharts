Method Chaining

PHP 5.3+ allows us to chain almost all aspects of LavaCharts

Provided are some examples of method chaining.

Configuration Objects

$titleStyle = Lava::textStyle()->color('#F3BB00')
                               ->fontName('Arial')
                               ->fontSize(20);
    
$backgroundColor = Lava::backgroundColor()->stroke('#113bc9')
                                          ->strokeWidth(4)
                                          ->fill('#ffd');

DataTables

$chartData = Lava::DataTable('Times')
                 ->addColumn('date', 'Dates', 'dates')
                 ->addColumn('number', 'Estimated Time', 'schedule')
                 ->addColumn('number', 'Actual Time', 'run')
                 ->addRow(array(Lava::jsDate(2013, 8, 1), 15, 23))
                 ->addRow(array(Lava::jsDate(2013, 8, 2), 12, 21))
                 ->addRow(array(Lava::jsDate(2013, 8, 3), 16, 20))
                 ->addRow(array(Lava::jsDate(2013, 8, 4), 13, 23))
                 ->addRow(array(Lava::jsDate(2013, 8, 5), 18, 19))

Charts

Lava::LineChart('Times')->title('Times for Deliveries')
                        ->fontSize(20)
                        ->fontName('Impact')
                        ->colors(array('blue', 'green', 'red'))
