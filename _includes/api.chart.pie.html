is3D( boolean $is3D )

If set to true, displays a three-dimensional chart

  • parambooleanIf TRUE, chart is 3D and disables some other options.
  • returnsobject (Chart)Returns the chart for chaining.

slices( array $slices )

An array of slice objects, each describing the format of the corresponding slice in the pie.
The values of the array keys will correspond to each numbered piece of the pie, starting from 0.
You can skip slices by assigning the keys of the array as (int)s.

  • paramarray of objs (slice)Array of objects, type slice.
  • returnsobject (Chart)Returns the chart for chaining.

pieSliceBorderColor( string $pieSliceBorderColor )

Sets the color of the slice borders. (only applies if is3D == [ FALSE | NULL ])

  • paramstringValid HTML color code.
  • returnsobject (Chart)Returns the chart for chaining.

pieSliceText( string $pieSliceText )

Sets the content of the text displayed on the slice.

  • paramstringValid values:
    "percentage" - The percentage of the slice size out of the total.
    "value" - The quantitative value of the slice.
    "label" - The name of the slice.
    "none" - No text is displayed.
  • returnsobject (Chart)Returns the chart for chaining.

pieSliceTextStyle( textStyle $pieSliceTextStyle )

Sets the text style of the slices.

  • paramobject (textStyle)The textStyle applied to the slice.
  • returnsobject (Chart)Returns the chart for chaining.

pieStartAngle( int $pieStartAngle )

Sets the angle, in degrees, to rotate the chart by.

  • paramintThe default of 0 will orient the leftmost edge of the first slice directly up.
  • returnsobject (Chart)Returns the chart for chaining.

reverseCategories( boolean $reverseCategories )

If set to true, will draw slices counterclockwise. The default is to draw clockwise.

  • paramboolean[ TRUE | FALSE ]
  • returnsobject (Chart)Returns the chart for chaining.

sliceVisibilityThreshold( numeric $sliceVisibilityThreshold )

The slice relative part, below which a slice will not show individually.
All slices that have not passed this threshold will be combined to a single slice, whose size is the sum of all their sizes.
Default is not to show individually any slice which is smaller than half a degree.

  • paramint | floatThreshold amount.
  • returnsobject (Chart)Returns the chart for chaining.

pieResidueSliceColor( string $pieResidueSliceColor )

Color for the combination slice that holds all slices below "sliceVisibilityThreshold".

  • paramstringValid HTML color string.
  • returnsobject (Chart)Returns the chart for chaining.

pieResidueSliceLabel( string $pieResidueSliceLabel )

A label for the combination slice that holds all slices below "sliceVisibilityThreshold".

  • paramstringSlice label.
  • returnsobject (Chart)Returns the chart for chaining.
