Lava API

The Lava Class is used to create & store DataTables and Charts
as well as creating configuration objects and placing the chart into the page.

  • See DataTables

Lava::XxxxChart( string $label )

Lava::div( string $elementID | int $width, int $height )

Builds a div html element with ID for a chart to be rendered into.

  • paramstringAssigns a label to the DataTable, and stores it in the Lava Class.
  • ORSet the dimensions as (int)s and the element ID is assigned automatically from the outputInto() method.
  • paramintSets CSS width for the div.
  • paramintSets CSS height for the div.
  • returnshtmlDiv HTML element to place in page for the chart to be rendered into.

Lava::hasErrors()

Checks if any errors were encountered while setting up the chart.

  • returnsbooleanTRUE if errors occured, otherwise FALSE.

Lava::getErrors()

Returns any errors that may have been encountered while configuring.
Errors are wrapped with the defined html tag that can be set in the "app\config\packages\khill\lavacharts\config.php".

See how to publish the LavaChart config file.
  • returnsstringError string[s], wrapped in html element.
